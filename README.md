## Cara Run Project

Pastikan Anda sudah menginstall Node.js dan aplikasi Expo Go di smartphone.

1.  **Clone Repository**
    ```bash
    git clone 
    cd 
    
    ```
2.  **Install Dependencies**
    ```bash
    npm install
    ```
3.  **Run Metro Bundler**
    ```bash
    npx expo start -c
    ```
4.  **Scan QR Code**: Buka aplikasi **Expo Go** di Android/iOS dan scan QR Code yang muncul di terminal.
